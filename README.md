# Voice AI Assistant

A multi-language AI chatbot web application built with Streamlit and Google Gemini API, featuring voice input capabilities and multiple AI personalities.

## About

This intelligent chatbot allows users to interact through both voice and text input. The AI can respond in English or Chinese (Mandarin) and adapts its personality based on user selection.

## Features

- **Voice Input & Output**: Record your voice and get automatic speech-to-text transcription with AI audio responses
- **Multi-Language Support**: Switch between English and Chinese (Mandarin) with language-aware AI responses
- **AI Personalities**: Choose from 4 different AI personalities:
  - **General Assistant** - Helpful and friendly for any questions
  - **Study Buddy** - Patient learning companion for educational topics
  - **Fitness Coach** - Energetic workout and nutrition guide
  - **Gaming Helper** - Expert gaming tips and strategies
- **Text Input**: Type messages manually as an alternative to voice input
- **Chat History**: View your conversation history with the AI

## Tech Stack

- Streamlit
- Google Gemini 2.5 Flash API
- Google Text-to-Speech (gTTS)
- Google Speech Recognition API
- Python
